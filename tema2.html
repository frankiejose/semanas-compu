los edificios hace tiempo atras se construian de piedra pero con el paso del tiempo los edificios se hiban haciendo mas altos y com eso la piedra dejo de ser la mejor opcion para construir edificios cada vez mas altos, si una persona quisiera construir un edificio de piedra alto tendria que construir los pisos de abajo del eificio con paredes muy gruesas haciendo que no se tuviera el suficiente espacio en ese piso. entonces se encontro una mejor forma de construir edificios y esta fue con ormigon que con unos moldes para hacer las paredes de cada piso que este molde se desarmaba y se volvia a armar cada piso, este preceso se realizaba hasta completar todos los pisos del edificio en constrccion.

Investigación

dd/mes/aa

para construir un deificio alto se encesita una base imprecionantemante firme para que el deificio no se caiga por los fuertes vientos, el peso de la estructura, la resistencia de los materiales o por algun otro motivo. los materiales utilizados para contruir edificios altos en ormigon con metal solido al rededor esto esta bajo tierra para soportar todo el edificio

los edificios altos tienen que tener formas aerodinamicas para que el aire no sea un problema para el edificio, el Burg kalifa tiene una forma de semicirculo en los costados del edificio para desviar las corrientes de aire, si esto no se hace el edificio se tambaleara de un lado al otro haciendo que en los pisos superiores sea impiocible ocuparlos, nunva veras a un edificio alto cuadrado totalmente esto por las fuertes corrientes de aire que gastaran la estructura y que el edifico se mueva de un lado al otro
